The File structure of Django mainly consists of:

1. `django-admin startproject` command.
2. Basic flow of Django:  
   user -> req -> urls (-> app urls) -> views  
   views (-> db, temp, direct) -> res -> user.
3. Templates, static.
4. Templating engines (load static, etc.).
