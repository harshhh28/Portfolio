**The File structure of Django mainly consists of:**<br />

<br />

1. `django-admin startproject` command.<br />
2. **Basic flow of Django:**<br />
   user -> req -> urls (-> app urls) -> views<br />
   views (-> db, temp, direct) -> res -> user.<br />
3. **Templates, static.**<br />
4. **Templating engines (load static, etc.).**<br />
